double_shift()¶

Performs an inefficient double shift for a real valued hessenberg matrix H with complex eigenvalues. Calculates the real matrix \(M = H^2 - 2\Re(\sigma)H + |\sigma|^2 I\) and the double shifted \(H_2 = Q^{T} H Q\), where \(Q\) is from the \(QR\) decomposition of \(M\).

static givens_22(x)¶

Generates an 2 \(\times\) 2 special Givens Rotation matrix based on the on the \(2 \times 1\) vector x. Function is useful for producing Givens matrices stabily and efficiently. For a Givens rotation matrix \(G\), vector \(u\)

\[\begin{split}G u = \left[ \begin{array}{c} u_1 \\ 0 \end{array} \right]\end{split}\]
Parameters

x (array) – \(2 \times 2\) vector who’s 2nd entry needs to be reduced to 0.

Returns

Return type

A \(2 \times 2\) Givens rotation matrix.

static givens_rotation(i, j, x, n)¶

Generates an n \(\times\) n special Givens Rotation matrix based on the parameters i, j, x, and n. The rotation matrix acts to reduce the j`th component of the vector `x to 0. For a Givens rotation matrix \(G\), vector \(u\) and index \(j\)

\[\begin{split}G u = \left[ \begin{array}{c} u_1 \\ \vdots \\ u_{j - 1} \\ 0 \\ u_{j - 1} \\ \vdots \\ u_{n} \\ \end{array} \right]\end{split}\]
Parameters

  • i (int) – `i`th row.

  • j (int) – `j`th column.

  • x (array) – Vector who’s `j`th entry needs to be reduced to 0.

  • n (int) – Size of the returned Givens Matrix.

Returns

A Givens rotation matrix.

Return type

numpy ndarray
