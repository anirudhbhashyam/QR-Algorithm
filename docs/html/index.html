Hessenberg¶

qr.hessenberg.complex_matrix(n, a, b)¶

Produces a random n \(\times\) n complex square matrix. All values in the matrices range between 2a and 2b.

Parameters

  • n (int) – Square matrix size.

  • a (float) – Lower limit for random number generator.

  • b (float) – Upper limit for random number generator.

Returns

A complex square matrix.

Return type

numpy ndarray

Raises

ValueError – If b \(\leq\) a.

qr.hessenberg.hessenberg_transform(M, calc_u=True)¶

Converts a given complex square matrix to Hessenberg form using Houeholder transformations.

Parameters

  • M (ndarray) – A complex square numpy 2darray.

  • calc_u (bool) – Flag to determine whether to calculate the transfomation matrix.

Return type

[<class ‘numpy.ndarray’>, <class ‘numpy.ndarray’>]

Returns

  • numpy ndarray – The transformed matrix

  • numpy ndarray – The permutation matrix if calc_q = True.

qr.hessenberg.householder_reflector(x)¶

Produces the Householder vector based on the input vector x. The householder vector \(u\) acts as

\[\begin{split}(I-2uu^{*})x = \left[ \begin{array}{c} \alpha \\ 0 \\ \vdots \\ 0 \\ \end{array} \right]\end{split}\]
Parameters

x (array) – A numpy array who’s entries after the 1st element needs to be 0ed.

Returns

The Householder vector.

Return type

numpy array
