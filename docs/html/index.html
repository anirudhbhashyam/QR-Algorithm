double_shift(eps: float, n_iter: int)¶

Performs an inefficient double shift for a real valued hessenberg matrix H with complex eigenvalues. Calculates the real matrix \(M = H^2 - 2\Re(\sigma)H + |\sigma|^2 I\) and the double shifted \(H_2 = Q^{T} H Q\), where \(Q\) is from the \(QR\) decomposition of \(M\). Produces the real schur form \(H = U^{*}RU\).

Parameters

n_iter – Number of double iterations to perform.

Returns

\(H_2\)

Return type

numpy ndarray

Raises

ValueError: – If self.H is not real.

static givens_rotation(i: int, j: int, x: numpy.array, n: int) → numpy.ndarray¶

Generates an n \(\times\) n special Givens Rotation matrix based on the parameters i, j, x, and n. The rotation matrix acts to reduce the j`th component of the vector `x to 0. For a Givens rotation matrix \(G\), vector \(u\) and index \(j\)

\[\begin{split}G u = \left[ \begin{array}{c} u_1 \\ \vdots \\ u_{j - 1} \\ 0 \\ u_{j - 1} \\ \vdots \\ u_{n} \\ \end{array} \right]\end{split}\]
Parameters

  • i – ith row.

  • j – jth column.

  • x – Vector who’s jth entry needs to be reduced to 0.

  • n – Size of the returned Givens Matrix.

Returns

A Givens rotation matrix.

Return type

numpy ndarray
