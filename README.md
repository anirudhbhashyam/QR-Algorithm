# Prerequisites
Running the prediction model requires these basic dependancies:

* Python `>=3.8.0`
* Numpy `>= 1.21.0`
* Pandas `>= 1.3.0`
* Sphinx (for building documentation) `>=4.3.0`.

Detailed environment information can be found in `requirements.txt`.
